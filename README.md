# ✨ Interface Glassmorphism

Projeto desenvolvido com HTML e CSS criando uma interface com efeito glassmorphism, com foco em design moderno e organização visual.

---

## 🛠️ Tecnologias utilizadas
- HTML
- CSS

---

## 🎯 Objetivo
Praticar a criação de interfaces visuais modernas, explorando efeitos como transparência, blur e uso de cores.

---

## 🌐 Visualizar projeto
https://liadaine.github.io/glassmorphism-ui/


---

## 📚 O que foi praticado
- Estruturação de layout com HTML  
- Estilização com CSS  
- Uso do efeito glassmorphism (blur e transparência)  
- Organização de elementos na interface  
- Criação de design moderno
